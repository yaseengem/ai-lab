The demo0 product is branded "AI Agents Squad" in the UI (launcher demo card, dashboard subtitle, agent list heading) and config. Although "Squad" reads as informal, the user explicitly wants it RETAINED as the brand name — they reversed a rename to "AI Agents" / "AI Agents Marketplace".

**How to apply:** Do not "professionalize" this name away during copy/UI polish. See [[ui-casing-restraint]] and the spec at specs/ui-professional-polish.md.
